
対応色だの人気投票結果だのとにらみ合いしつつ
独断と偏見で選んだ16人の東方キャラスキンです。
野生状態では浮遊する陰陽玉みたいな感じです、ちょっと不気味。

1.8対応後、陰陽玉から出現するキャラクターがランダムになりました。
